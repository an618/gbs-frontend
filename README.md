# Goalbase

## Install Node.js (install LTS version)

```
https://nodejs.org/en
```

## Check Version

```
node --version
npm --version
```

## Install Angular CLI

```
npm install -g @angular/cli
```

## Install the dependencies

```
npm install
```

## Run the Angular Project

```
npm run start
```

### Every Commit Message Contain

```
build ready
```

## Add .env file

```
NG_APP_BASE_URL=localhost
```
